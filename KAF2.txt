Installation steps:
1) verify if java is installed on machine. Run command, java -version. If not, refer to following link to install java.
	https://www.tutorialspoint.com/apache_kafka/apache_kafka_installation_steps.htm
2) Zookeeper Installation and configuring:
   1) visit the link and download (ZooKeeper-3.4.6.tar.gz) : http://zookeeper.apache.org/releases.html 
   
   2) extract the tar file: 
      cd /opt
	  tar -zxf zookeeper-3.4.6.tar.gz
	  cd zookeeper-3.4.6
	  mkdir data
   
   3) create a configuration file: Open conf/zoo.cfg, add/edit the following as,
		tickTime=2000
		dataDir=/hadoop/zookeeper/data
		clientPort=2181
		initLimit=5
		syncLimit=2
		autopurge.snapRetainCount=30
   
   4) once the config file is set, try to start zookeeper-server from the location where it is placed,
       example, 
       bin/zkServer.sh start
	   
	   O/p: it should read the config file and show,
	   $ Starting zookeeper ... STARTED
   
   5) zookeeper cli can be started as : bin/zkCli.sh
   
   6) Stop Zookeeper Server: bin/zkServer.sh stop
   
   Zookeeper is used to maintain a kafka broker cluster.
   
3) Kafka Installation and configuring: 
   1) visit this link and download: https://www.apache.org/dyn/closer.cgi?path=/kafka/0.9.0.0/kafka_2.11-0.9.0.0.tgz 
   
   2) extract tar file:
		$ cd opt/
		$ tar -zxf kafka_2.11.0.9.0.0 tar.gz
		$ cd kafka_2.11.0.9.0.0
		
   3) check the config property : server.properties
   
   4) start the server: bin/kafka-server-start.sh config/server.properties
   
   5) stop the server: $ bin/kafka-server-stop.sh config/server.properties
