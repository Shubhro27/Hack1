Python came up with Matplotlib and Seaborn are 2 packages which build visualizations seamlessly (and is faster).
 Currently, Matplotlib ver. 1.5.1 is the stable version available.

Components of a Matplotlib visualization:
1) Figure: window/page where the diagram is drawn
2) Axes: area on which the data is plotted (with Matplotlib functions). Each Axes has a x-axis and y-axis
3) Spines: lines that designate the boundaries of the data area.
4) Subplots: used to set up and place axes on a regular grid.i.e. in case of multiple curves to be designed within the same axes.

Artist Object: Everything drawn using Matplotlib is part of the Artist module.

Matplotlib has 2 major modules:
1) Pyplot: allows programmer to implicitly create figures and axes to achieve desired plot.
2) Pylab : this imports both (pyplot and numpy libs) and recommended when working with arrays and mathematical functions.

Basic operations:
1) plot() : creates object and outputs info on it. This does not show the figure created.
			If plot() is given only a single array/list,
			a) assumes it is a vector of y-values
			b) Automatically generates a x-vector of same length as Y axis with consecutive integes starting with ZERO.
				it usually give the number of rows in a dataframe.
			
			Plot(x,y) function helps pass both x and y vectors and len(x) == len(y)
			
2) show() : displays figure.

example,

	# create the 1st sub-plot for sine wave

Histograms using matplotlib:
histogram is a graphical representation of the frequency distribution of data. 
In Histogram:
Bars           : Rectangles of equal horizontal size corresponding to class interval. i.e. X-Axis
Variable Height: Corresponds to frequency. i.e. Y-Axis

numpy.histogram() function is used to plot histogrms using Python. It takes the "INPUT ARRAY" and "BINS" as two parameters.
example,
	import numpy as np 
	#   
	a = np.array([22,87,5,43,56,73,55,54,11,20,51,5,79,31,27]) 
	hist,bins = np.histogram(a,bins = [0,20,40,60,80,100]) 
	print hist 
	print bins

	o/p:
	[3 4 5 2 1]    
	[0 20 40 60 80 100]
	
What happened i.e. how to calculate frequency (Y-axis)
STEP-1: sort the array (grouped values)
        5	5	11	20	22	27	31	43	51	54	55	56	73	79	87
STEP-2: determine the bins i.e. from above program [0,20,40,60,80,100]. 
		This can be grouped into following categories,
		0-20
		20-40
		40-60
		60-80
		80-100
STEP-3: now count which elements in the array (STEP-1) fall in these categories,
		0-20	3
		20-40	4
		40-60	5
		60-80	2
		80-100	1
STEP-4: so [3,4,5,2,1] are the frequencies.

More on Frequency distribution: https://www.mathsisfun.com/data/frequency-distribution-grouped.html
								Detailed: https://math.tutorvista.com/statistics/frequency-distribution.html
								
Frequency Distribution is usually plotted via Bar-graph, Histogram, pie-chart.

Scatter Plot:
 Scatter plots is used when comparing variables, for example, how much one variable is affected by another variable to build a relation out of it.  
 The data is displayed as a collection of points.
 
 example,
	import matplotlib.pyplot as plt
	x = [1,1.5,2,2.5,3,3.5,3.6]
	y = [7.5,8,8.5,9,9.5,10,10.5]
	# 
	x1=[8,8.5,9,9.5,10,10.5,11]
	y1=[3,3.5,3.7,4,4.5,5,5.2]
	# 
	plt.scatter(x,y, label='high income low saving',color='r')
	plt.scatter(x1,y1,label='low income high savings',color='b')
	plt.xlabel('saving*100')
	plt.ylabel('income*1000')
	plt.title('Scatter Plot')
	plt.legend()
	plt.show()
	
	o/p: The data is displayed as a collection of points having ‘high income low salary’ and ‘low income high salary’.

Area Plot:
These plots can be used to track changes over time for two or more related groups that make up one whole category.
example, work done during a day into categories, say sleeping, eating, working and playing.

example,
	import matplotlib.pyplot as plt
	#
	days = [1,2,3,4,5]
	#  
	sleeping =[7,8,6,11,7]
	eating = [2,3,4,3,2]
	working =[7,8,7,2,2]
	playing = [8,5,7,8,13]
	#
	plt.plot([],[],color='m', label='Sleeping', linewidth=5)
	plt.plot([],[],color='c', label='Eating', linewidth=5)
	plt.plot([],[],color='r', label='Working', linewidth=5)
	plt.plot([],[],color='k', label='Playing', linewidth=5)
	#
	plt.stackplot(days, sleeping,eating,working,playing, colors=['m','c','r','k'])
	#
	plt.xlabel('x')
	plt.ylabel('y')
	plt.title('Stack Plot')
	plt.legend()
	plt.show()
